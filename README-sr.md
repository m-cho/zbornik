# Зборник - Православна хришћанска апликација

**[🇺🇸 English](README.md)**


**Зборник** је православна мобилна и веб апликација (PWA) за свакодневну молитву, читање Светог Писма и праћење литургијског календара. Омогућава приступ молитвенику, више превода Библије и основним литургијским ресурсима на српском и енглеском језику у једноставном интерфејсу.

## Функционалности (реализовано)
- 📖 **Читање Светог Писма** — православни преводи: српски (екавски/ијекавски) и црквенословенски.
- 🙏 **Молитвеник (sr-Cyrl)** — молитвеник на српској ћирилици.
- 📱 **Вишеплатформска подршка** — PWA и покретање преко Expo за Android.
- 🌙 **Тема** — подршка за тамну и светлу тему.

## Коришћене технологије
- React Native + Expo
- TypeScript
- Zustand (State Management)
- Expo Router (Navigation)

## Покретање

Захтеви:

- Node.js (препоручено v20)
- npm (или yarn / pnpm)
- Expo CLI (опционо глобално: `npm install -g expo-cli`)

Брзи почетак:

```bash
# Инсталирај зависности
npm install

# Покрени развојни сервер (Metro + Expo)
npm start

# Покрени на Android емулатору или уређају
npm run android

# Покрени као веб апликацију (PWA)
npm run web
```

Развојни ток рада:

- Креирај feature гране и пошаљи pull request за промене.
- Тестирај локално преко `npm start` и провери на Android или вебу.
- Остави TypeScript типове и lint чистим; прати постојећи стил кода.

Корисне команде:

- `npm run build` — (ако постоји) изгради production bundle.
- `npm test` — покрени тестове (ако их има).
- `npm run lint` — покрени ESLint.

Допринoшење:

- Fork-уј репо, направи грана тему и отворени pull request.
- PR-ови нека буду фокусирани и садрже кратак опис промена.
- За веће промене, прво отвори issue ради договора о дизајну.

Где тражити:

- Улаз апликације и руте: app/app.tsx и фолдер app/app/
- Компоненте: components/
- Ресурси (молитвеник, фонтови, слике): app/assets/ и app/prayer-book/
