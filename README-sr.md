# Зборник - Православна хришћанска апликација

**[🇺🇸 English](README.md)**

**Зборник** је православна хришћанска мобилна апликација и PWA дизајнирана да подржи духовни живот и свакодневну молитвену праксу. Апликација пружа приступ основним православним текстовима и литургијским ресурсима на више језика.

## Функционалности
- 📖 **Читање Светог Писма** - Више православних превода Библије
- 🙏 **Молитвеник** - Збирка православних молитава
- 📅 **Литургијски календар** - Јеванђељска поглавља текуће недеље и празници
- 🌐 **Подршка за више језика** - Српски (ћирилица/латиница) и енглески
- 📱 **Вишеплатформска** - PWA и нативна Android подршка
- 🌙 **Тамна/светла тема** - Прилагодљиво искуство читања

## Планиране функционалности
- 📜 **Охридски пролог** - Дневна духовна читања Св. Николаја Велимировића
- ⛪ **Литургијски текстови** - Божанствена литургија и службени текстови
- 🔍 **Напредна претрага** - Претрага кроз сав садржај
- 🔖 **Обележивачи** - Чување омиљених одломака и молитава

## Коришћене технологије
- React Native + Expo
- TypeScript
- Zustand (State Management)
- Expo Router (Navigation)

## Покретање

```bash
# Инсталирај зависности
npm install

# Покрени развојни сервер
npm start

# Android
npm run android

# Веб
npm run web
```
