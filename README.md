# Zbornik - Orthodox Christian App

**[🇷🇸 Srpski / Serbian](README-sr.md)**

**Zbornik** is an Orthodox Christian mobile application and PWA designed to support spiritual life and daily prayer practice. The app provides access to essential Orthodox texts and liturgical resources in multiple languages.

### Features
- 📖 **Holy Scripture Reading** - Multiple Orthodox Bible translations
- 🙏 **Prayer Book** - Complete collection of Orthodox prayers
- 📅 **Liturgical Calendar** - Current week Gospel readings and feast days
- 🌐 **Multi-language Support** - Serbian (Cyrillic/Latin) and English
- 📱 **Cross-platform** - PWA and native Android support
- 🌙 **Dark/Light Theme** - Customizable reading experience

### Planned Features
- 📜 **Ohrid Prologue** - Daily spiritual readings by St. Nikolai
- ⛪ **Liturgical Texts** - Divine Liturgy and service texts
- 🔍 **Advanced Search** - Full-text search across all content
- 🔖 **Bookmarks** - Save favorite passages and prayers

### Technology Stack
- React Native + Expo
- TypeScript
- Zustand (State Management)
- Expo Router (Navigation)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start

# Android
npm run android

# Web
npm run web
```
